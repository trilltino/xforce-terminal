# Reference Materials

This directory contains reference documentation and code examples.

**Note:** The actual content is not tracked in git. Clone the repositories as needed:

```bash
# Rust web frameworks
git clone https://github.com/tokio-rs/axum.git ref/axum
git clone https://github.com/async-graphql/async-graphql.git ref/async-graphql
git clone https://github.com/rust-web-app/rust-web-app.git ref/rust-web-app

# GTK4 Rust bindings
git clone https://github.com/gtk-rs/gtk4-rs.git ref/gtk4-rs
git clone https://github.com/gtk-rs/gtk-rs-core.git ref/gtk-rs-core

# Stellar/Soroban
git clone https://github.com/stellar/rs-soroban-sdk.git ref/rs-soroban-sdk
git clone https://github.com/stellar/rs-soroban-client.git ref/rs-soroban-client
git clone https://github.com/stellar/stellar-docs.git ref/stellar-docs
git clone https://github.com/stellar/go.git ref/stellar-go
git clone https://github.com/stellar/js-stellar-sdk.git ref/stellar-sdk
git clone https://github.com/stellar/soroban-examples.git ref/soroban-examples

# Horizon API docs (static HTML)
# Download from https://developers.stellar.org/docs/data/horizon
```
