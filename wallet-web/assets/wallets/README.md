# Wallet Logo Images

Place wallet logo images in WebP format here:

- `phantom.webp` - Phantom wallet logo
- `solflare.webp` - Solflare wallet logo  
- `backpack.webp` - Backpack wallet logo
- `sollet.webp` - Sollet wallet logo (if needed)

Recommended size: 32x32px or 64x64px for best quality.

You can download wallet logos from:
- Phantom: https://phantom.app/
- Solflare: https://solflare.com/
- Backpack: https://www.backpack.app/

Convert to WebP format using an online converter or image tool.
